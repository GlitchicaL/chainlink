# Chainlink Testing & Examples

Examples of Chainlink features such as contract automation, and random numbers using VRF.

## Technology Stack & Tools

- Solidity (Smart Contracts)
- TypeScript (Scripts & Testing)
- [Hardhat](https://hardhat.org/) (Development Framework)
- [Ethers.js](https://docs.ethers.io/v5/) (Blockchain Interaction)
- [Alchemy](https://www.alchemy.com/) (Testnet Connection)
- [Chainlink](https://chain.link/)

## Requirements For Initial Setup
- [NodeJS](https://nodejs.org/en/). Ideally best using NodeJS via [NVM](https://github.com/nvm-sh/nvm#intro).
- An [Alchemy](https://www.alchemy.com/) account.

## Setting Up
### 1. Clone/Download the Repository

### 2. Install Dependencies:
`npm install`

### 2. Install Dependencies:
`npx hardhat test`
